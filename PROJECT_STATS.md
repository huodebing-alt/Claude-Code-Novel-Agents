## Tech stack

- **Agents/skills**: Markdown + YAML frontmatter (Claude Code native)
- **Orchestrator**: Python 3.10+, standard library only (no pip deps), `urllib` for API calls
- **Outline Reviewer**: Python `http.server` (stdlib) + single-page HTML + vanilla JS + SortableJS (CDN)
- **PDF**: pandoc + WeasyPrint + Chromium-headless + wkhtmltopdf (whichever is present) + browser "Save as PDF" fallback
- **Typography**: Cormorant Garamond · Noto Serif SC · Inter
- **Palette**: ivory `#f7f2e7` · oxblood `#7a1f1f` · matte gold `#c9a96b` · charcoal `#2c2c2c`
